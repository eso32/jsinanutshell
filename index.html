Selecting elements

  • document.getElementById('id')
  • document.getElementsByTagName('tag') returns collection
  • document.getElementsByClassName('class') returns collection
  • document.querySelector('.someClass')
  • document.querySelectorAll('p:nth-child(3n+1)') returns collection

Modifying the DOM

  • element.textContent('New text')
  • element.innerHTML('New HTML code')
  • element.('New HTML code')

Styling elements

  • element.style.color = 'red'
  • element.style.backgroundColor = 'green'
    • CamelCase transitions CSS: background-color becomes JS: backgroundColor

Adding and removing elements

  • document.createElement('div')
  • ParentElement.appendChild('childElement')
  • ParentElement.removeChild('childElement')
